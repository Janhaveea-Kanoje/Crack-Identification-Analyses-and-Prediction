Introduction

Business Problem - Damage in infrastructure can raise serious safety concerns and yet is very necessory to kept in a adequate condition. however, inspecting them manually is very time consuming and needs a lot of labor source. By using this pipeline, we can analyze and supervise different surface conditions in a time-effective and cost-effective way.

Data source for this model is: https://www.kaggle.com/datasets/arunrk7/surface-crack-detection/data

This idea can be formulated in a machine learning task.

Problem statement - Develop Deep learning model that can identify cracks in infrastucture and identify positive or negative. (positive = cracks; negative = no cracks)

def image_samples(dataset, class_names):
    batch = next(iter(dataset))
    images, labels = batch

    fig, axes = plt.subplots(3, 3, figsize=(8, 8))
    for i, ax in enumerate(axes.flat):
        ax.imshow(images[i].numpy().astype("uint8"))
        ax.set_title(class_names[labels[i]])
        ax.axis('off')

    plt.tight_layout()
    plt.show()

Preprocessing

Scale the image dataset, to avoid gradient decent issues and improve the model performance.

Use one hot encoding to avoid none value errors.

In[84]:
def scale_img(image, label):
  image = image / 255.0

  label = _tf_.one_hot(label, depth=len(class_names))
  return image, label

scaled_image_set = image_set.map(scale_img)

model = keras.Sequential()

model.add(layers.Rescaling(1./255, input_shape=(src_height, src_width, 3)))

model.add(layers.LeakyReLU(alpha=0.1))
model.add(layers.BatchNormalization())
model.add(layers.MaxPooling2D())

model.add(layers.Conv2D(32, (5, 5), padding="same"))
model.add(layers.LeakyReLU(alpha=0.1))
model.add(layers.BatchNormalization())
model.add(layers.MaxPooling2D())

model.add(layers.Conv2D(32, (5, 5), padding="same"))
model.add(layers.LeakyReLU(alpha=0.1))
model.add(layers.BatchNormalization())
model.add(layers.MaxPooling2D())

model.add(layers.Conv2D(64, (5, 5), padding="same"))
model.add(layers.LeakyReLU(alpha=0.1))
model.add(layers.BatchNormalization())
model.add(layers.MaxPooling2D())

model.add(layers.Flatten())

model.add(layers.Dense(256))
model.add(layers.LeakyReLU(alpha=0.1))
